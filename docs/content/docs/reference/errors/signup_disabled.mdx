---
title: signup_disabled
description: Signup disabled error
---

This error occurs when sign-up is disabled for an auth flow and a user tries to create a new account through that flow.

## How to fix

If you want to block sign-up based on application policy, use a database hook so the decision is enforced server-side.

```ruby title="config/auth.rb"
auth = BetterAuth.auth(
  secret: ENV.fetch("BETTER_AUTH_SECRET"),
  database_hooks: {
    user: {
      create: {
        before: ->(user, ctx = nil) {
          unless SignupPolicy.allowed?(user)
            raise BetterAuth::APIError.new(
              "BAD_REQUEST",
              message: "Signup is disabled"
            )
          end

          user
        }
      }
    }
  }
)
```
