---

This example mounts Better Auth Ruby directly in a Rack application.

**Implements the following features:**
Email & Password . Social Sign-in . Session Management

## Install

```ruby title="Gemfile"
source "https://rubygems.org"

gem "rack"
gem "puma"
gem "better_auth"
gem "pg"
```

```bash
bundle install
```

## Configure Auth

```ruby title="config/auth.rb"
require "better_auth"
require "better_auth/adapters/postgres"
require "better_auth/social_providers"

DATABASE_URL = ENV.fetch("DATABASE_URL")

Auth = BetterAuth.auth(
  secret: ENV.fetch("BETTER_AUTH_SECRET"),
  base_url: ENV.fetch("BETTER_AUTH_URL", "http://localhost:9292"),
  database: BetterAuth::Adapters::Postgres.new(
    url: DATABASE_URL
  ),
  email_and_password: {
    enabled: true
  },
  social_providers: {
    google: BetterAuth::SocialProviders.google(
      client_id: ENV.fetch("GOOGLE_CLIENT_ID"),
      client_secret: ENV.fetch("GOOGLE_CLIENT_SECRET")
    ),
    discord: BetterAuth::SocialProviders.discord(
      client_id: ENV.fetch("DISCORD_CLIENT_ID"),
      client_secret: ENV.fetch("DISCORD_CLIENT_SECRET")
    )
  }
)
```

## Mount Rack

```ruby title="config.ru"
require_relative "./config/auth"

run Rack::URLMap.new(
  "/api/auth" => Auth,
  "/" => ->(_env) {
    [
      200,
      { "content-type" => "text/plain" },
      ["Better Auth Ruby is running"]
    ]
  }
)
```

## Run

```bash
bundle exec rackup
```

The auth endpoints are available under `/api/auth`.
